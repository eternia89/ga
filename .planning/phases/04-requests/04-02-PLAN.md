```typescript
type TimelineEvent = {
  type: 'created' | 'status_change' | 'triage' | 'field_update' | 'rejection' | 'cancellation';
  at: string;       // ISO timestamp
  by: string;       // user email or name
  details?: Record<string, unknown>;  // varies by type

```

Render a vertical timeline (left-aligned):

  - `created`: Plus icon (green)
  - `status_change`: ArrowRight icon (blue)
  - `triage`: ClipboardCheck icon (blue)
  - `field_update`: Pencil icon (gray)

  - `cancellation`: Ban icon (stone/gray)

  - "Created" — "{user} submitted this request" + timestamp

  - "triage" — "{user} triaged this request" + show changed fields (Category: X, Priority: Y, PIC: Z) + timestamp
  - "field_update" — "{user} updated {field}" + old/new values + timestamp
  - "rejection" — "{user} rejected this request" + reason text (blockquote style) + timestamp
  - "cancellation" — "{user} cancelled this request" + timestamp
- Timestamps formatted as `dd-MM-yyyy, HH:mm:ss` (MANDATORY date+time format)
