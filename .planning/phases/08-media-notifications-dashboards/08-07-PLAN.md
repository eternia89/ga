1. Install ExcelJS: `npm install exceljs`

2. Create `lib/exports/excel-helpers.ts`:
   - Import ExcelJS types
   - Export `applyStandardStyles(sheet: ExcelJS.Worksheet)`:
     - Header row (row 1): bold, white text (argb: 'FFFFFFFF'), blue background (argb: 'FF4472C4'), centered alignment, height 24
     - All cells: thin borders (top, left, bottom, right)
     - Auto-fit columns: calculate width based on header text length + 4, with minimum 10
   - Export `createStyledWorkbook(sheetName: string, columns: { header: string, key: string, width: number }[]): { workbook, sheet }`:
     - Create workbook, add worksheet with frozen header row (`views: [{ state: 'frozen', ySplit: 1 }]`)
     - Set columns
     - Return workbook and sheet references
   - Export `generateExcelResponse(workbook: ExcelJS.Workbook, filename: string): Response`:
     - Call `workbook.xlsx.writeBuffer()`
     - Return NextResponse with buffer, Content-Type `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`, Content-Disposition `attachment; filename="${filename}"`
   - Note: ExcelJS requires Node.js runtime. Do NOT set `export const runtime = 'edge'` in any export route.

3. Create `app/api/exports/requests/route.ts`:
   - GET endpoint
   - Auth check (getUser, profile, check role — only ga_lead, admin, finance_approver can export)
   - Fetch ALL requests (no filter, export everything per user decision): `select('*').is('deleted_at', null).order('created_at', { ascending: false })`
   - Entity-only columns per user decision — NO denormalized related data (no job info in request export):
     - ID (display_id), Description, Status (mapped via STATUS_LABELS), Priority (mapped via PRIORITY_LABELS), Location ID, Category ID, Requester ID, Assigned To, Created At (dd-MM-yyyy format), Updated At
   - Use `createStyledWorkbook`, add rows, `applyStandardStyles`, `generateExcelResponse`
   - Filename: `requests-export-{dd-MM-yyyy}.xlsx`

4. Create `app/api/exports/jobs/route.ts`:
   - Same pattern as requests
   - Auth check
   - Fetch ALL jobs: `select('*').is('deleted_at', null).order('created_at', { ascending: false })`
   - Columns: ID (display_id), Title, Description, Status, Priority, Assigned To, Created By, Created At, Updated At
   - Filename: `jobs-export-{dd-MM-yyyy}.xlsx`

   - When `exportUrl` is provided, render an Export button (Download icon from lucide-react) at the right end of the toolbar
   - Export button: `<Button variant="outline" size="sm">` with Download icon + "Export" text
   - On click: trigger browser download via `window.location.href = exportUrl` (simplest approach for file download) or use `fetch` + `blob` + `createObjectURL` approach for better UX:
     ```
     const handleExport = async () => {
       setExporting(true);
       try {
         const response = await fetch(exportUrl);
         const blob = await response.blob();
         const url = URL.createObjectURL(blob);
         const a = document.createElement('a');
         a.href = url;
         a.download = response.headers.get('content-disposition')?.split('filename=')[1]?.replace(/"/g, '') || 'export.xlsx';
         a.click();
         URL.revokeObjectURL(url);
       } finally {
         setExporting(false);
       }
     };
     ```
